# Classical Machine Learning Projects

This repository contains a collection of **10 classical Machine Learning projects** built using:

- **NumPy**
- **Pandas**
- **Matplotlib**
- **Scikit-Learn**

The goal of this repo is to master the **foundations of ML** before moving into Deep Learning and PyTorch.  
Each project includes complete workflows:
- Data loading  
- Cleaning & preprocessing  
- Feature engineering  
- Model training  
- Evaluation  
- Visualizations
